## Installation

ZHKWebSource is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'ZHKWebSource', :git => 'https://github.com/Ruris/ZHKWebSource.git'
```
